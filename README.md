# Soren Hundertmark's CV Project

## Description
A single-page HTML CV showcasing my experience, skills, and education as a Junior Software Engineer. (p.s. Highly recommend using Roadmap.sh's project to build foundational skills)

## Features
- Responsive design
- Clean, professional layout
- Favicon support
- SEO meta tags and Open Graph tags for social media sharing

## Technologies Used
- HTML
- CSS (minimal: inline styling)

## How to View
1. Clone or download this repository
2. Open `cv-project.html` in your web browser
3. Or use VS Code's Go Live extension for live preview
